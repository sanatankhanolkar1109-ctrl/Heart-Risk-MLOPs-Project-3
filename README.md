Heart Risk Prediction - MLOps Project
## Overview

This project predicts the risk of heart disease using a Machine Learning model deployed through FastAPI and containerized using Docker.

The objective of the project is to demonstrate an end-to-end Machine Learning deployment workflow, covering model training, serialization, API development, containerization, and production-ready serving.

## Problem Statement

Heart disease remains one of the leading causes of mortality worldwide.

This project leverages Machine Learning techniques to estimate the likelihood of heart disease based on patient health indicators and clinical measurements.

The model can assist in identifying high-risk patients and support early medical intervention.
